# Triangle Calculations

## Why this exists

This project explores triangle related logic such as checking valid sides or computing area or perimeter.

## What this project shows

It shows how I combine user input, conditionals, and math formulas to solve small geometry problems in C.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
