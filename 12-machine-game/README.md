# Vending Machine Game

## Why this exists

A console-based vending machine style game that simulates a simple interactive system.

## What this project shows

It demonstrates menu driven programs, state management, input checking, and separating UI from core logic with a library file.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
