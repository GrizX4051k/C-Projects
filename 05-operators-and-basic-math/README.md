# Operators and Basic Math

## Why this exists

These exercises focus on arithmetic, relational, and logical operators plus basic math checks like even or odd.

## What this project shows

Each program shows control flow with if and else, operator precedence, and how to turn simple formulas into working C code.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
