# Student Database

## Why this exists

This project builds a small student records system to store and manage basic student information.

## What this project shows

It demonstrates structs, arrays of records, simple CRUD operations, and experimenting with a more modular library design.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
