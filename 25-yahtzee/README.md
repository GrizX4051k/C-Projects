# Yahtzee Game

## Why this exists

A console version of the Yahtzee dice game, built as a larger C project.

## What this project shows

It shows work with arrays, probability style logic, scoring functions, and splitting a bigger project across multiple source and header files.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
