# Factorial

## Why this exists

This project exists to practice implementing factorial using loops or recursion.

## What this project shows

It reinforces my understanding of iteration, recursion, and overflow considerations when working with integers.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
