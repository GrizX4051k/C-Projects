# Linked List

## Why this exists

A linked list implementation with basic operations like insert, delete, and traverse.

## What this project shows

It demonstrates dynamic memory management with malloc and free, pointer manipulation, and building a core data structure in C.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
