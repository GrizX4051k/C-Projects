# Year and Date

## Why this exists

This project exists to practice working with year based logic like leap year checks or simple date calculations.

## What this project shows

It shows conditional logic, modular arithmetic, and turning rules (like leap year rules) into C code.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
