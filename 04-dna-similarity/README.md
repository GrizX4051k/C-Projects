# DNA Similarity

## Why this exists

This project exists to explore basic bioinformatics ideas by comparing two DNA strings and computing a similarity score.

## What this project shows

It demonstrates string processing, looping over characters, and designing a simple scoring algorithm in C.

## How to run (local)

Open this folder in your terminal and compile with GCC or your C compiler.

Single program example:
  gcc main.c -o app -mconsole
  ./app   (or .\app.exe on Windows)

For folders with multiple small programs:
  gcc file_name.c -o file_name.exe -mconsole
  .\file_name.exe
